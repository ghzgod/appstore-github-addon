####App Store GitHub Addon###
Adds GitHub star counts to every Community Applications app tile, plus a sortable GitHub view of the whole catalog — by stars, trending (day/week/month/year), or newest to the App Store. It uses CA's own app page and never modifies the Community Applications plugin.

Set your GitHub personal access token at **Settings → Utilities → App Store GitHub Addon**. Star data is cached in appdata and refreshed automatically.
